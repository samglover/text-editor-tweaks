A super-simple WordPress plugin that improves both tabs of the WordPress editor with better fonts and typography.


== Description ==

WP Editor Tweaks is a super-simple WordPress plugin that improves both tabs of the WordPress editor with better fonts from Google Fonts ([Source Sans Pro](https://www.google.com/fonts/specimen/Source%20Sans%20Pro) in the *Visual* editor and [Source Code Pro](https://www.google.com/fonts/specimen/Source%20Code%20Pro) in the *Text* editor) and typography. Once you start using WP Editor Tweaks, you may even *like* drafting your posts in WordPress!


== Installation ==

Install WP Editor Tweaks from *Plugins > Add New* in your WordPress install or by uploading the files to your server.
