# WP Editor Tweaks for WordPress

## Description

WP Editor Tweaks is a super-simple WordPress plugin that improves both tabs of the WordPress editor with better fonts and typography. Once you start using WP Editor Tweaks, you may even *like* drafting your posts in WordPress!
